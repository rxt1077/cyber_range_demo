Challenge 3

From a backend perspecting, this is the most complicated type of challenge yet. Instead of using a simple web page or a single container, it uses a group of containers via Docker Compose and allows the participant to VPN into the internal network. This simulates a full environment, but it's going to be pretty hefty in terms of compute requirements. You probably wouldn't want to do it for every challenge.
Use the following WireGuard configuration to connect to the network environment for this challenge:
{{ wgconf }}
This environment will remain active for 60 minutes
This will give you the IP address 10.3.13.37 on the container network (10.3.13.0/24). Once you are connected, find the hostname of the "stealth" host on the network. Its hostname is the flag.
HINT: aznc vf lbhe sevraq
